### Sub-Collections

**Note**: Shopify doesn't natively support nested collections. Use one of these approaches:

**Option A: Tags-Based Filtering**
- Tag products with `gold` or `silver`
- Create automated collections: "Gold Bracelets" (products tagged `bracelets` AND `gold`)
- Collection handles: `gold-bracelets`, `silver-bracelets`, `gold-chains`, `silver-chains`

**Option B: Separate Collections**
- Create separate collections: "Gold Bracelets", "Silver Bracelets", etc.
- Assign products directly to these collections
- Use collection handles: `gold-bracelets`, `silver-bracelets`, `gold-chains`, `silver-chains`

**Recommended**: Use Option B (separate collections) for clearer organization and easier frontend filtering.
