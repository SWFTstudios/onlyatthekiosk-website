About

8xFlow is a styleguide made to implement individual UI designs with Webflow as fast as never before.

The advantage over other styleguides is that it combines the 8 point spacing system with a 12 column grid.

No more guessing which class name stands for what. No matter how many projects you're working on at the same time or how long a project has been on hold, the class naming is meaningful with maximum flexibility for implementing individual UIs.
